Xiaomi Poco F1 6.18 Inch 4G LTE Smartphone

The Streaming Blog
August 30, 2018

Xiaomi Poco F1 Smartphone comes with a 6.18-inch notched display, Snapdragon 845 processor, LiquidCool technology, 4,000mAh battery, and more. The phone features dual rear cameras, backed by Artificial Intelligence. The primary sensor is 12MP, while the secondary lens is a 5MP one. The front camera is 20MP with AI Beautify feature. The smartphone runs MIUI and it supports face unlock feature as well.


Package Contents:

  * 1 x Xiaomi Poco F1 Smartphone
  * 1 x Type-C Cable
  * 1 x User Manual
  * 1 x Power Adapter

Package Details:

  * Package Size: 19.00 x 9.00 x 6.00 cm / 7.48 x 3.54 x 2.36 inches
  * Package Weight: 0.3600 kg

Product Features:

  * 6.18-inch 2246*1080 Pixel FHD+ screen
  * CPU: Qualcomm Snapdragon 845, Octa core, 2.8GHz, GPU: Adreno 630
  * MIUI Base on Android OS , 6GB RAM + 64GB ROM
  * Dual Nano SIM card dual standby
  * Camera Type: Front camera:20.0MP, Back camera:12.0MP+5.0MP
  * Support FDD LTE/4G+/3G/GPS/Bluetooth/Miracast/OTG/Type-C/Touch ID/IR Face unlock

Networking:

  * 2G GSM B2/3/5/8
  * 3G WCDMA B1/2/5/8
  * 4G FDD-LTE B1/3/5/7/8/20, TDD-LTE B38/40/41(120MHz)
